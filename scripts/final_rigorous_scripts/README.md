## How to Use

### Run All Experiments for One Dataset
```bash
cd rigorous_per_dataset/BasicMotions
bash run_all_methods.sh
```

### Run All Experiments Across All Datasets
```bash
for dataset_dir in rigorous_per_dataset/*/; do
  dataset=$(basename "$dataset_dir")
  cd "$dataset_dir"
  bash "${dataset}_all_methods.sh"
  cd ../..
done
```
